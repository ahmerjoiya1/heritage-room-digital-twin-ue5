Prototype Description
Purpose

This prototype explores how Unreal Engine 5 can be used to create an interactive heritage-room digital twin environment. The aim is to understand the practical software engineering challenges involved in constructing cultural heritage spaces as navigable and visually meaningful digital environments.

Prototype Scope

The current prototype includes an interior heritage-style room, environmental lighting, exterior surroundings, and day/night scene conditions. It is not intended to be a historically verified reconstruction of a specific site. Instead, it serves as an exploratory artefact for understanding how heritage environments can be structured, visualised, and extended toward research-oriented digital twin systems.

Main Components

The prototype currently includes:

A heritage-style interior room
Period-inspired furniture and objects
Exterior landscape elements
Day lighting condition
Night lighting condition
Walkthrough videos for both lighting modes
Screenshots documenting the visual environment
